# docs/ — Documentação do TCC

Local para o documento final (ABNT), referências, plano de ação e materiais auxiliares.

## Sugestão de subdivisão (criar conforme necessário)

```
docs/
├── tcc/             # Documento principal (ABNT) — LaTeX ou docx
├── referencias/     # Papers, relatórios técnicos baixados
├── plano-acao/      # Cronograma detalhado, divisão de tarefas, atas de reunião
├── apresentacoes/   # Slides de qualificação e defesa
└── diagramas/       # Diagramas de arquitetura, fluxos, mockups
```

## Convenções

- Documento final em formato ABNT (a equipe escolhe LaTeX ou Word)
- Referências em BibTeX se LaTeX
- Datas em formato ISO (YYYY-MM-DD)
